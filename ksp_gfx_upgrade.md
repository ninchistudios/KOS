# About

This is a step by step guide to switch a stock KSP RSS/RO/RP-1 Express Install (High Graphics) to use Sol and Blackrack's volumetric clouds.

* Clouds (Blackrack 14 Mar 26): https://www.patreon.com/posts/release-5-139881534
* Video (Goddchen 18 Apr 26): https://www.youtube.com/watch?v=Tc_BBQx_Eu8
* Goddchen's Modlist: https://pastebin.com/tig434sA

# Starting Point

The video begins the same as a stock Express RSS/RO/RP-1 install, except using Sol graphics config instead of RO. 

* Add the Sol repo to CKAN as per the video

At this point with a fresh install we'd use High Graphics during the Express install, but for Sol we'd choose Sol Core. We've already installed High, so we need to remove that and install Sol Core instead via CKAN.
